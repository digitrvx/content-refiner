# Content Refiner ✍️✨
**A Professional, High-Performance Bulk Text Removal Tool**

Content Refiner is a lightweight, privacy-focused web application designed to help editors, developers, and content creators clean up large blocks of text by removing specific unwanted phrases or multiple lines instantly.

[Go to Live](https://digitrvx.github.io/content-refiner/) ---

## 🚀 Key Features
- **Bulk Phrase Removal:** Delete multiple unique phrases or lines at once.
- **Instant Processing:** Handles large datasets in milliseconds using optimized JavaScript.
- **Privacy Guaranteed:** All processing is done locally in your browser. No data is sent to any server.
- **Responsive Design:** Fully functional on Desktops, Tablets, and Mobile devices.
- **Power User Friendly:** Supports `Ctrl + Enter` (or `Cmd + Enter`) for quick submission.

## 🛠️ How It Works
Content Refiner uses a **Global Regular Expression (Regex)** algorithm to scan your main content and strip out every occurrence of your specified "blacklist" phrases.

1. **Input Phrases:** Enter the words or lines you want to remove in the top box (one per line).
2. **Paste Content:** Paste your large text in the main body area.
3. **Refine:** Click "Refine Text Now" or press `Ctrl + Enter`.
4. **Copy:** Use the built-in copy button to grab your clean text.
